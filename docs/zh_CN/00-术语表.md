# 00. Yggdrasil v3 术语表

> 本术语表用于统一中文与英文文档中的核心概念。代码标识符不翻译。

| English | 中文建议 | 说明 |
|---|---|---|
| App | App / 应用实例 | 框架运行时的应用级入口 |
| App-local runtime | App 本地运行时 | 不依赖进程全局状态的运行时 |
| process-default facade | 进程默认兼容门面 | 面向 legacy / 单主 App 的兼容入口 |
| Module Hub | Module Hub / 模块中心 | 管理长期模块和 capability 的容器 |
| Module | Module / 模块 | 长期存在的能力载体 |
| Capability | Capability / 能力 | typed contract，用于模块间发现 |
| assembly.Spec | `assembly.Spec` | 公共声明式计划，不翻译 |
| prepared runtime assembly | prepared runtime assembly / 已准备运行时装配图 | 内部实例图，不作为公共 API |
| BusinessBundle | `BusinessBundle` | 业务组合阶段的正式输出 |
| Runtime narrow surface | Runtime 窄接口 | 业务 Compose 阶段可访问的安全框架入口 |
| Staged Reload | 分阶段热重载 / Staged Reload | prepare / commit / rollback reload 协议 |
| restart-required | 需要重启 | 当前变更不能安全热更新 |
| degraded | 降级状态 | reload rollback 失败或状态分叉 |
| provider | provider / 提供者 | 提供具体能力或工厂的对象 |
| resolver watch | resolver watch | 动态服务发现监听对象，不进 Hub |
